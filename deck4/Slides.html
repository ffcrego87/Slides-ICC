Properties of Regular Languages

There are severe limits to the recognition problems that an FA can handle.

But this requires proof.

Closure Properties

A set is closed under an operation if applying that operation to any members of the set always yields a member of the set.

  • Fact: The set of regular languages is closed under each Kleene operation.

That is, if \$L_1\$ and \$L_2\$ are regular languages, then each of \$L_1 \cup L_2\$, \$L_1 L_2\$ and \$L_1^{\ast}\$ is regular.

Proving Closure under Kleene

The easiest approach is to show that the REs for \$L_1\$ and \$L_2\$ can be combined or adjusted to form the RE for the combination language.

Example: The RE for \$L_1 L_2\$ is obtained by writing down the RE for \$L_1\$ followed by the RE for \$L_2\$.

Closure under Complementation

Fact: The set of regular languages is closed under complementation.

The complement of language \$L\$, written \$\overline{L}\$, is all strings not in \$L\$ but with the same alphabet.

The statement says that if \$L\$ is a regular language, then so is \$\overline{L}\$.

To see this fact, take a deterministic FA for \$L\$ and interchange the accept and reject states.

Closure under Intersection

Fact: The set of regular languages is closed under intersection.

One approach: Use de Morgan’s law:

\$L_1 \cap L_2 = \overline{(\overline{L_1} \cup \overline{L_2})}\$

and that regular languages are closed under union and complementation.

Product Construction for Intersection

Each state in the product is a pair of states from the original machines.

Formally, if

  • \$L_1\$ is accepted by DFA \$M_1\$ with 5‑tuple \$(Q_1,\ \Sigma,\ q_1,\ T_1,\ \delta_1)\$ and

  • \$L_2\$ is accepted by DFA \$M_2\$ with 5‑tuple \$(Q_2,\ \Sigma,\ q_2,\ T_2,\ \delta_2)\$,

then \$L_1 \cap L_2\$ is accepted by the DFA

\$(Q_1 \times Q_2,\ \Sigma,\ (q_1,q_2),\ T_1 \times T_2,\ \delta)\$ where \$\delta((r,s),x) = (\delta_1(r,x),\ \delta_2(s,x))\$.

  • \$L_1\$ is the binary strings with an even number of \$0\$’s, and

  • \$L_2\$ the binary strings with an even number of \$1\$’s.

A Nonregular Language: \$0^n 1^n\$

There is no FA for

\$B = \{0^n 1^n : n \ge 0\} = \{\varepsilon, 01, 0011, 000111, \ldots\}\$

Informal argument: Suppose input is known to be \$0\$’s followed by \$1\$’s. The FA has to count the \$0\$’s, i.e., at the end of the \$0\$’s it must be in a state unique to the number of \$0\$’s read.

But an FA has only fixed finite memory, while there can be arbitrarily many \$0\$’s.

Impossible!

(In)Distinguishable Strings

Two strings \$x\$ and \$y\$ are indistinguishable with respect to language \$L\$

if for every string \$z\$, it holds that \$xz \in L\$ iff \$yz \in L\$.

Otherwise they are distinguishable.

Example: Let \$A\$ be the language of binary strings with odd number of \$1\$’s. Then strings 0 and 10001 are indistinguishable with respect to \$A\$.

Distinguishable Strings and States

Theorem: If \$M\$ is a DFA accepting language \$L\$, and \$x\$ and \$y\$ are distinguishable strings with respect to \$L\$, then \$M\$ must be in a different state after reading \$x\$ than after reading \$y\$.

Suppose strings \$x\$ and \$y\$ put \$M\$ in the same state. Then for any string \$z\$, the strings \$xz\$ and \$yz\$ put \$M\$ in the same state. So either both in \$L\$ or both out. Hence \$x\$ and \$y\$ are indistinguishable.

Sets of Distinguishable Strings

Corollary: If \$D_L\$ is a set of pairwise distinguishable strings with respect to \$L\$, then any DFA for \$L\$ has at least \$|D_L|\$ states.

In particular, if \$D_L\$ is infinite then \$L\$ is not regular.

Example \$0^n 1^n\$

Recall \$B=\{0^n 1^n : n \ge 0\}\$.

The set \$D_B = \{0^j : j \ge 0\}\$ is pairwise distinguishable.

Take any two strings in \$D_B\$: say \$0^j\$ and \$0^{j'}\$ with \$j \ne j'\$. Appending \$1^j\$ to the first produces a string in \$B\$, but appending \$1^j\$ to the second produces a string not in \$B\$. Thus they are distinguishable.

Since \$D_B\$ is infinite, \$B\$ is not regular.

Example: Lower Bound on Number of States

Consider the language of all binary strings with an even number of both \$0\$’s and \$1\$’s. The set \$\{\varepsilon, 0, 1, 01\}\$ is pairwise distinguishable.

So the product‑construction DFA above has the fewest states possible.

(In general, there is a connection between the smallest number of states of a DFA and the largest set of pairwise distinguishable strings.)

Practice

Define \$E\$ as the language of all binary strings with an equal number of \$0\$’s and \$1\$’s.

Show that \$E\$ is not regular by finding an infinite set of pairwise distinguishable strings.

Solution

Let \$D_E\$ be the set of all strings containing only \$0\$’s.

Then for \$i \ne j\$ the strings \$0^i\$ and \$0^j\$ are distinguishable with respect to \$E\$, since \$0^i 1^i \in E\$, but \$0^i 1^j \notin E\$.

The set \$D_E\$ is infinite, hence \$E\$ is nonregular.

The Pumping Lemma

The infamous Pumping Lemma says that every regular language has a certain repetitiveness.

It’s a tool to prove that a language is not regular: it provides a property that all regular languages must satisfy.

The Pumping Lemma

Let \$A\$ be a regular language accepted by a DFA with \$k\$ states. Then, for any string \$z \in A\$ with at least \$k\$ symbols, we can split \$z = uvw\$ where:

  • \$v\$ is nonempty,

  • \$|uv| \le k\$, and

  • \$uv^i w \in A\$ for all \$i \ge 0\$.

Using the Pumping Lemma

To show nonregularity, choose one string \$z\$ that does not pump and derive a contradiction.

Example: \$0^n 1^n\$

Consider \$B = \{0^n 1^n : n \ge 0\}\$.

Suppose \$B\$ were regular. Then some DFA with \$k\$ states accepts it.

Take \$z = 0^k 1^k \in B\$. In any split \$z=uvw\$ with \$|uv| \le k\$, the piece \$v\$ contains only \$0\$’s, so pumping down gives \$uw\$ with fewer \$0\$’s than \$1\$’s, not in \$B\$. Contradiction.

Example: Palindromes

A palindrome reads the same backwards and forwards.

Let \$P\$ be all palindromes over \$\{a,b\}\$. \$P\$ is nonregular.

If \$P\$ were regular, there is a DFA with \$k\$ states. Take \$z=a^k b a^k\$. Any split with \$|uv| \le k\$ has \$v\$ made of \$a\$’s; pumping breaks the palindrome. Contradiction.

Practice

Define \$E\$ as the language of all binary strings with an equal number of \$0\$’s and \$1\$’s.

Show nonregularity via the Pumping Lemma.

Practice Solutions

Assume \$E\$ regular with DFA of \$k\$ states. Take \$z=0^k 1^k\$. As \$|uv| \le k\$, \$v\$ is only \$0\$’s. Pumping yields unequal counts, so \$uw \notin E\$. Contradiction.

Javascript

Let’s design a finite automaton (FA) that accepts binary strings with an even number of \$0\$’s and an even number of \$1\$’s.

Javascript – Steps for Implementation

  1. States/parities: AX (even 0, even 1) accept; AY (even 0, odd 1); BX (odd 0, even 1); BY (odd 0, odd 1).

  2. Transitions: Reading 0 toggles the 0‑parity; reading 1 toggles the 1‑parity.

Javascript – Code

class FiniteAutomaton {
  constructor() {
    this.state = 'AX';
    this.acceptingStates = new Set(['AX']);
  }
  transition(input) {
    switch (this.state) {
      case 'AX': this.state = (input === '0') ? 'BX' : 'AY'; break;
      case 'AY': this.state = (input === '0') ? 'BY' : 'AX'; break;
      case 'BX': this.state = (input === '0') ? 'AX' : 'BY'; break;
      case 'BY': this.state = (input === '0') ? 'BX' : 'AY'; break;
    }
  }
  isAccepted(s) {
    this.state = 'AX';
    for (let c of s) this.transition(c);
    return this.acceptingStates.has(this.state);
  }
}

Javascript – NFA Example

An NFA (nondeterministic FA) can have multiple transitions per symbol, and epsilon transitions.

Example NFA that accepts strings containing substring 01.

class NFA {
  constructor() {
    this.transitions = {
      q0: { '0': ['q1'], '1': ['q0'] },
      q1: { '0': ['q1'], '1': ['q2'] },
      q2: { '0': ['q2'], '1': ['q2'] } // sink-accept
    };
    this.startState = 'q0';
    this.acceptingStates = new Set(['q2']);
  }
  isAccepted(s) {
    let current = new Set([this.startState]);
    for (let c of s) {
      const next = new Set();
      for (let st of current) {
        const outs = (this.transitions[st] || {})[c] || [];
        for (let t of outs) next.add(t);
      }
      current = next;
    }
    for (let st of current) if (this.acceptingStates.has(st)) return true;
    return false;
  }
}

Overview (recap)

A regular language is one with an FA or an RE. Regular languages are closed under union, concatenation, star, intersection, and complementation.

To prove nonregularity, use infinite sets of pairwise distinguishable strings or the Pumping Lemma.
