Regular Expressions

A regular expression (RE) describes a language.

It uses the three regular operations, called union/or, concatenation and star.

Brackets ( and ) are used for grouping, just as in normal math.

Union

The symbol \$+\$ means union or or.

Example:
\$0 + 1\$ means either a zero or a one.

Concatenation

The concatenation of two REs is obtained by writing one after the other.

Example:
\$(0 + 1) 0\$ corresponds to {00, 10}

\$(0 + 1)(0 + \epsilon)\$ corresponds to {00, 0, 10, 1}

Star

The symbol \$\ast\$ is pronounced "star" and means zero or more copies.

Example:
\$a^{\ast}\$ corresponds to { \$\epsilon\$, a, aa, aaa, … }

\$(0+1)^{\ast}\$ corresponds to all binary strings.

Example

An RE for the language of all binary strings of length at least 2 that begin and end in the same symbol:

\$0(0+1)^{\ast}0 + 1(0+1)^{\ast}1\$

Precedence note: star applies to the smallest possible piece.

Example

Consider the regular expression:

\$((0+1)^{\ast} 1 + \epsilon)(00)^{\ast} 00\$

This is the set of all binary strings that end with an even, nonzero number of zeros.

Different language to: \$(0+1)^{\ast} (00)^{\ast} 00\$

Regular Operators for Languages

If RE is formed by or of REs \$R\$ and \$S\$, result is \$L(R) ∪ L(S)\$.

If RE is formed by concatenation of \$R\$ and \$S\$, result is all strings formed by concatenating one from \$R\$ and one from \$S\$.

If RE is formed by star of \$R\$, result is all strings formed by concatenating zero or more strings from \$R\$.

Regular Operators Example

If \$L = {ma, pa}\$ and \$M = {be, bop}\$,

  • \$L+M = {ma, pa, be, bop}\$

  • \$LM = {mabe, mabop, pabe, pabop}\$

  • \$L^{\ast} = {\epsilon, ma, pa, mama, …, pamamapa, …}\$

Notation: If \$\Sigma\$ is an alphabet, \$\Sigma^{\ast}\$ is the set of all strings over \$\Sigma\$.

An RE for Decimal Numbers

English: “Some digits followed maybe by a point and some more digits.”

Regular Expression:
\$(- + \epsilon) D D^{\ast} (\epsilon + . D^{\ast})\$ where \$D\$ is a digit.

Kleene’s Theorem

There is an FA for a language if and only if there is an RE for the language.

Proof (to come) is algorithmic.

A regular language is one accepted by some FA or described by an RE.

Applications of REs

  • Specify part of a programming language, e.g., real number, allows automated tokenizer generation.

  • Complex search and replace.

  • Many UNIX commands accept REs.

Exercise

Give an RE for each of these languages:

  1. All binary strings with at least one 0

  2. All binary strings with at most one 0

  3. All binary strings starting and ending with 0

In each case, several answers are possible.

Exercise

Give an RE for each of these languages:

  1. All binary strings with at least one 0 \$(0+1)^{\ast} 0 (0+1)^{\ast}\$

  2. All binary strings with at most one 0 \$1^{\ast} + 1^{\ast} 0 1^{\ast}\$

  3. All binary strings starting and ending with 0 \$0 (0+1)^{\ast} 0 + 0\$

In each case, several answers are possible.

Summary

A regular expression (RE) is built from symbols using the Kleene operators:

  • union (+)

  • concatenation

  • star (\$\ast\$)

The star of a language includes all concatenations of its strings (possibly repeated); the empty string is always included.
